Introdução

Este é um exemplo de página com design simples e responsivo, utilizando estilos integrados e boas práticas de desenvolvimento.

HTML

HTML é a linguagem de marcação usada para estruturar o conteúdo da web, como textos, imagens e links.

CSS

CSS é usado para estilizar e definir o layout de páginas web, controlando cores, fontes e espaçamentos.

JavaScript

JavaScript é uma linguagem de programação que adiciona interatividade e dinamismo às páginas web.
